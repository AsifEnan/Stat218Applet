# 2-Sample Proportions Confidence Interval

Constructs a confidence interval for the difference between two
population proportions (\\\pi_1 - \pi_2\\) using one of three methods:
the 2SD simulation method, a general simulation method, or a
theory-based approach.

This function accepts input in **two ways**:

- **Summary Statistics:** You already know the number of successes and
  sample sizes for both groups. Pass `success_1`, `n_1`, `success_2`,
  and `n_2` directly. Note that `method = "2SD"` and
  `method = "simulation"` are not available with summary statistics.

- **Raw Data:** You have an actual dataset loaded into R. Pass
  `formula`, `data`, and `success_level` instead, and the function
  counts successes for you automatically. All three methods are
  available with raw data.

## Usage

``` r
ci_2prop(
  success_1 = NULL,
  n_1 = NULL,
  success_2 = NULL,
  n_2 = NULL,
  group_names = NULL,
  formula = NULL,
  data = NULL,
  success_level = NULL,
  conf_level = 0.95,
  method = "2SD",
  sim_reps = 1000
)
```

## Arguments

- success_1:

  A whole number representing the count of successes in Group 1. For
  example, if 45 out of 100 patients recovered, `success_1 = 45`. **Only
  used when NOT providing `formula` and `data`.**

- n_1:

  The sample size of Group 1. **Only used when NOT providing `formula`
  and `data`.**

- success_2:

  A whole number representing the count of successes in Group 2. **Only
  used when NOT providing `formula` and `data`.**

- n_2:

  The sample size of Group 2. **Only used when NOT providing `formula`
  and `data`.**

- group_names:

  A character vector of length 2 giving meaningful names to the two
  groups. For example, `group_names = c("Treatment", "Control")`.
  Defaults to `c("Group 1", "Group 2")`. When using raw data, group
  names are extracted automatically from the grouping variable. **Only
  used when NOT providing `formula` and `data`.**

- formula:

  A two-sided formula of the form `Response ~ Group` identifying the
  categorical response variable and the grouping variable in your
  dataset. For example, `formula = Recovered ~ Treatment`. **Only used
  when NOT providing summary statistics.**

- data:

  A data frame (loaded from a `.csv`, `.txt`, or `.xlsx` file)
  containing the variables named in `formula`. **Only used when NOT
  providing summary statistics.**

- success_level:

  A character string specifying which value of the response variable
  counts as a "success". For example, if the response variable contains
  `"Yes"` and `"No"`, use `success_level = "Yes"`. **Required when using
  `formula` and `data`.**

- conf_level:

  The desired confidence level as a decimal. Default is `0.95` for a 95%
  confidence interval. Common choices are `0.90`, `0.95`, and `0.99`.
  Note that `method = "2SD"` only works at `0.95`.

- method:

  The method used to construct the confidence interval. Must be one of:

  `"2SD"`

  :   (default) The 2SD Simulation Method. An independent bootstrap is
      generated by resampling with replacement from each group
      separately.The interval is \\(\hat{p}\_1 - \hat{p}\_2) \pm 2
      \times SD\\. Only valid for 95% confidence interval limit. If you
      need a different confidence level, use `method = "simulation"`.

  `"simulation"`

  :   The Bootstrap Simulation Method. Same independent bootstrap as
      2SD, but uses the middle `conf_level`% of the simulated
      distribution to find the bounds. Works for any confidence level.

  `"theory"`

  :   The Theory-Based Method. Uses the unpooled standard error formula
      and the appropriate Z\* multiplier. Works for any confidence level
      and with both summary statistics and raw data.

- sim_reps:

  The number of bootstrap samples when `method = "2SD"` or
  `method = "simulation"`. Default is `1000`. Increasing to `5000` gives
  a more stable interval.

## Value

An S3 object of class `stat218_2prop_ci` containing all computed values.
You can use the following methods on the result:

- `print(result)`:

  Displays a clean summary of the interval including both group
  proportions, the observed difference, SD of sampling distribution, and
  the interval bounds.

- `plot(result)`:

  Shows the bootstrap or theoretical distribution with the confidence
  interval shaded and labeled, a forest plot of the interval, and a
  beautified 2x2 contingency table.

- `plot_steps(result)`:

  Displays a detailed three-panel step-by-step construction of the
  interval with proper fraction notation.

## Details

### The Core Idea

A two-sample proportion confidence interval gives a range of plausible
values for the true difference \\\pi_1 - \pi_2\\. A positive interval
suggests Group 1 has a higher proportion; a negative interval suggests
Group 2 is higher; an interval containing zero is consistent with no
difference between the two groups.

### The 2SD Method

An independent bootstrap where successes and failures from each group
are resampled with replacement separately. The difference in simulated
proportions is recorded for each repetition. The SD of those differences
estimates the SE, and the interval is point estimate \\\pm 2 \times
SD\\. Note that unlike `test_2prop` which uses a permutation approach,
the CI bootstrap resamples each group independently – this is correct
for confidence intervals where we are not assuming the null hypothesis.

### Theory Method

Uses the **unpooled** standard error (unlike the hypothesis test which
uses a pooled proportion). This is because for a confidence interval we
are not assuming the two proportions are equal: \$\$SE =
\sqrt{\frac{\hat{p}\_1(1-\hat{p}\_1)}{n_1} +
\frac{\hat{p}\_2(1-\hat{p}\_2)}{n_2}}\$\$

### Validity Conditions for Theory Method

All four cells in the 2x2 contingency table must have at least 10
observations:

- Successes in Group 1 \\\geq 10\\

- Failures in Group 1 \\\geq 10\\

- Successes in Group 2 \\\geq 10\\

- Failures in Group 2 \\\geq 10\\

## Examples

``` r
# --- Summary Statistics (theory, default) ---
result <- ci_2prop(
  success_1 = 45, n_1 = 100,
  success_2 = 25, n_2 = 100,
  group_names = c("Treatment", "Control"),
  method = "theory"
)
print(result)
#> 
#> ── 95% Confidence Interval for Difference in Proportions (Theory) ──────────────
#> ℹ Treatment: p-hat1 = 0.45 (successes = 45, n = 100)
#> ℹ Control: p-hat2 = 0.25 (successes = 25, n = 100)
#> ℹ Point Estimate (p-hat1 - p-hat2): 0.2
#> ℹ SD of Sampling Distribution: 0.066
#> • Interval: (0.0707, 0.3293)
if (FALSE) { # \dontrun{
plot(result)
plot_steps(result)
} # }

# --- Summary Statistics (theory, 90% CI) ---
result2 <- ci_2prop(
  success_1 = 45, n_1 = 100,
  success_2 = 25, n_2 = 100,
  group_names = c("Treatment", "Control"),
  conf_level = 0.90, method = "theory"
)
print(result2)
#> 
#> ── 90% Confidence Interval for Difference in Proportions (Theory) ──────────────
#> ℹ Treatment: p-hat1 = 0.45 (successes = 45, n = 100)
#> ℹ Control: p-hat2 = 0.25 (successes = 25, n = 100)
#> ℹ Point Estimate (p-hat1 - p-hat2): 0.2
#> ℹ SD of Sampling Distribution: 0.066
#> • Interval: (0.0915, 0.3085)
if (FALSE) { # \dontrun{
plot(result2)
plot_steps(result2)
} # }

# --- Raw Data (2SD method) ---
car_data <- mtcars
car_data$transmission <- ifelse(mtcars$am == 1, "Manual", "Automatic")
car_data$engine <- ifelse(mtcars$vs == 1, "Straight", "V-shaped")
result3 <- ci_2prop(
  formula = transmission ~ engine, data = car_data,
  success_level = "Manual", method = "2SD"
)
#> ✔ Data extracted from "transmission" ~ "engine":
#> • V-shaped: 6 successes out of 18
#> • Straight: 7 successes out of 14
#> ℹ Success level: "Manual"
print(result3)
#> 
#> ── 95% Confidence Interval for Difference in Proportions (2sd) ─────────────────
#> ℹ V-shaped: p-hat1 = 0.3333 (successes = 6, n = 18)
#> ℹ Straight: p-hat2 = 0.5 (successes = 7, n = 14)
#> ℹ Point Estimate (p-hat1 - p-hat2): -0.1667
#> ℹ SD of Sampling Distribution: 0.1717
#> • Interval: (-0.5101, 0.1768)
if (FALSE) { # \dontrun{
plot(result3)
plot_steps(result3)
} # }

# --- Raw Data (simulation, 90% CI) ---
result4 <- ci_2prop(
  formula = transmission ~ engine, data = car_data,
  success_level = "Manual", conf_level = 0.90,
  method = "simulation"
)
#> ✔ Data extracted from "transmission" ~ "engine":
#> • V-shaped: 6 successes out of 18
#> • Straight: 7 successes out of 14
#> ℹ Success level: "Manual"
print(result4)
#> 
#> ── 90% Confidence Interval for Difference in Proportions (Simulation) ──────────
#> ℹ V-shaped: p-hat1 = 0.3333 (successes = 6, n = 18)
#> ℹ Straight: p-hat2 = 0.5 (successes = 7, n = 14)
#> ℹ Point Estimate (p-hat1 - p-hat2): -0.1667
#> ℹ SD of Sampling Distribution: 0.1653
#> • Interval: (-0.4365, 0.119)
if (FALSE) { # \dontrun{
plot(result4)
plot_steps(result4)
} # }

# --- Raw Data (theory) ---
result5 <- ci_2prop(
  formula = transmission ~ engine, data = car_data,

#> ✔ Data extracted from "transmission" ~ "engine":

#> ℹ Point Estimate (p-hat1 - p-hat2): -0.1667
#> ℹ SD of Sampling Distribution: 0.1738
#> • Interval: (-0.5073, 0.174)

#>   observations.
```
